MAJOR PROJECTS

STORAGE MANAGEMENT

Multi tenant SAAS Product to manage storage facilities and units. Based on microservice based architecture. Tech Stack: Azure, C#, .Net Core, Angular, MS Sql and Kubernetes

STOR-EDGE (DEPLOYMENT PROCESS)

Developer uses IDE such as Visual Studio to commit changes to Git repository. Git triggers a new build on Brigade pipelines. It packages microservices as containers and pushes them to the Azure Container Registry. Containers are deployed to AKS cluster. Users access services via apps and website. Azure Active Directory is used to secure access to different Azure resources. Every Microservices has different databases. Administrator accesses via a separate admin portal.

AZURE MIGRATION PROJECT

Large enterprise grade project for a university in the United States. Complex infrastructure with multiple servers migrated on Azure Cloud. Lift and shift strategy was adopted for quick turnaround. Designed infrastructure for fault tolerant system with auto scaling to optimize performance and costs
